Prerequisites:
--------------

- Install Python 3
- Install numpy
- Install keras (version 2.2.4)
- Install tensorflow (version 1.12.0)
- Install feedparser (only required for the feedScraper.py)


Note: You can also install keras-gpu and tensorflow-gpu if you want to run the model 
on the GPU instead of CPU


Instructions:
-------------

To train the model run 
'python lstm_text_gen.py'

To generate text using a previously trained model run:
'python generate_text.py
